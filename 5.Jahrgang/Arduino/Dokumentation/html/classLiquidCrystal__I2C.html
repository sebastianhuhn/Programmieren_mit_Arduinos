Ausführliche Beschreibung

This is the driver for the Liquid Crystal LCD displays that use the I2C bus.

After creating an instance of this class, first call begin() before anything else. The backlight is on by default, since that is the most likely operating mode in most cases.

Dokumentation der Elementfunktionen

void LiquidCrystal_I2C::begin ( )

Set the LCD display in the correct begin state, must be called before anything else is done.

void LiquidCrystal_I2C::blink ( )

Start blinking the cursor indicator.

void LiquidCrystal_I2C::clear ( )

Remove all the characters currently shown. Next print/write operation will start from the first position on LCD display.

void LiquidCrystal_I2C::cursor ( )

Show a cursor indicator, cursor can blink on not blink. Use the methods blink() and noBlink() for changing cursor blink.

void LiquidCrystal_I2C::display ( )

Show the characters on the LCD display, this is the normal behaviour. This method should only be used after noDisplay() has been used.

void LiquidCrystal_I2C::home ( )

Next print/write operation will will start from the first position on the LCD display.

void LiquidCrystal_I2C::noBlink ( )

Do not blink the cursor indicator.

void LiquidCrystal_I2C::noCursor ( )

Do not show a cursor indicator.

void LiquidCrystal_I2C::noDisplay ( )

Do not show any characters on the LCD display. Backlight state will remain unchanged. Also all characters written on the display will return, when the display in enabled again.
